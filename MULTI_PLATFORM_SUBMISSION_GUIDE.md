## 1️⃣ 必应Webmaster Tools（必应搜索）

### 提交地址
https://www.bing.com/webmasters/

### 详细步骤

#### 第一步：登录或注册
1. 访问 https://www.bing.com/webmasters/
2. 使用Microsoft账号登录（Outlook/Hotmail等）
3. 如果没有账号，先注册一个Microsoft账号

#### 第二步：添加站点
1. 点击"添加站点"（Add your site）
2. 输入：**alisnart.cn**
3. 点击"添加"

#### 第三步：验证站点所有权
**推荐方式：HTML文件验证**

1. 选择"HTML文件验证"（HTML file verification）
2. 下载必应提供的验证文件（类似 `BingSiteAuth.xml`）
3. **把验证文件发给开心**，我会帮您上传到服务器根目录
4. 回到必应Webmaster Tools，点击"验证"（Verify）

**备选方式：HTML标签验证**

1. 选择"HTML标签验证"（HTML meta tag verification）
2. 复制必应提供的meta标签
3. 添加到网站首页的 `<head>` 部分
4. 点击"验证"

#### 第四步：提交Sitemap
验证通过后：
1. 进入"Sitemaps"部分
2. 点击"提交Sitemap"（Submit sitemap）
3. 输入：`https://alisnart.cn/sitemap.xml`
4. 点击"提交"

#### 第五步：手动提交关键页面
1. 进入"URL提交"（URL Submission）部分
2. 逐个提交以下URL：
   - https://alisnart.cn/
   - https://alisnart.cn/index.html
   - https://alisnart.cn/about.html
   - https://alisnart.cn/services.html
   - https://alisnart.cn/cases.html
   - https://alisnart.cn/news.html

## 2️⃣ 谷歌Search Console（谷歌搜索）

### 提交地址
https://search.google.com/search-console/

### 详细步骤

#### 第一步：登录或注册
1. 访问 https://search.google.com/search-console/
2. 使用Google账号登录（Gmail等）
3. 如果没有账号，先注册一个Google账号

#### 第二步：添加属性
1. 点击"添加属性"（Add property）
2. 选择"网址前缀"（URL prefix）
3. 输入：**https://alisnart.cn**
4. 点击"继续"（Continue）

#### 第三步：验证站点所有权
**推荐方式：HTML文件验证**

1. 选择"HTML文件"（HTML file）
2. 下载谷歌提供的验证文件（类似 `google1234567890abcdef.html`）
3. **把验证文件发给开心**，我会帮您上传到服务器根目录
4. 回到Google Search Console，点击"验证"（Verify）

**备选方式：DNS记录验证**

1. 选择"DNS记录"（DNS record）
2. 复制谷歌提供的TXT记录
3. 在域名DNS管理后台添加该TXT记录
4. 等待DNS生效（可能需要几小时）
5. 回到Google Search Console，点击"验证"

#### 第四步：提交Sitemap
验证通过后：
1. 进入"Sitemaps"部分
2. 在"添加新的Sitemap"框中输入：`sitemap.xml`
3. 点击"提交"（Submit）

#### 第五步：使用URL检查工具
1. 进入"URL检查工具"（URL Inspection Tool）
2. 输入关键URL，点击"测试实时URL"（Test Live URL）
3. 点击"请求编入索引"（Request Indexing）

建议提交的URL：
- https://alisnart.cn/
- https://alisnart.cn/index.html
- https://alisnart.cn/about.html
- https://alisnart.cn/services.html
- https://alisnart.cn/cases.html
